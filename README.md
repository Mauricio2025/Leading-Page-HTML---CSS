# Meu Portfólio

Bem-vindo ao meu repositório de portfólio! Este projeto foi desenvolvido usando HTML, CSS, JavaScript, Node.js e Bootstrap. Aqui está uma breve descrição do projeto e instruções para configurar e executar localmente.

## Visão Geral

Este projeto consiste em um portfólio pessoal que exibe minhas habilidades, projetos e informações de contato de maneira organizada e estilizada. Foi desenvolvido utilizando as seguintes tecnologias:

- **HTML:** Estruturação da página.
- **CSS:** Estilização e layout.
- **JavaScript:** Adiciona interatividade e dinamismo à página.
- **Node.js:** Utilizado para configuração de servidor local (opcional).
- **Bootstrap:** Framework de CSS utilizado para design responsivo.

## Pré-requisitos

Certifique-se de ter o Node.js instalado em seu sistema antes de prosseguir.
